# 🚀 PetFlow: Gestão Inteligente para Pet Shops

[![Project Status: Active](https://img.shields.io/badge/Project%20Status-Active-green.svg)](https://www.repostatus.org/#active)
[![Academic Project](https://img.shields.io/badge/Context-Academic--PI--2026.1-blue.svg)](https://github.com/topics/projeto-integrador)

O **PetFlow** é um sistema de gestão inteligente para Pet Shops, focado em mitigar perdas financeiras por *no-show* e garantir conformidade com a LGPD. Desenvolvido como Projeto de Extensão para a disciplina de **Projeto Integrador (SENAC 2026.1)**.

## 🛠️ Stack Tecnológica
- **Linguagem:** Python 3.x
- **Banco de Dados:** SQLite3
- **Interface:** CLI (Command Line Interface)
- **Testes:** PyTest

## 📁 Estrutura do Projeto
- `src/`: Código fonte do sistema.
- `tests/`: Suíte de testes automatizados.
- `docs/`: Documentação acadêmica e técnica.
- `data/`: Localização do banco de dados SQLite (em desenvolvimento).
